# 2D_Node_Arranger

A node that you can use to arrange node in certain patterns. Useful for UI elements, cards for a card game, etc

## 📥 Installation
### Linux / MacOS
Copy and paste these commands into terminal at the root of your project:
```bash
clone-gd-addon https://github.com/ChillCube/2d_node_arranger.git
```

Full documentation can be found in [DOCUMENTATION.md](./DOCUMENTATION.md)
